# Winai School

โปรเจกต์เว็บแอปพลิเคชันระบบบริหารจัดการโรงเรียน Winai School พัฒนาด้วย Next.js สำหรับรองรับการใช้งานของนักเรียน ครู และผู้อำนวยการ

---

## วิธีเริ่มต้นใช้งาน

ติดตั้ง dependencies ก่อน

```bash
npm install
```

จากนั้นรันโปรเจกต์ด้วยคำสั่ง

```bash
npm run dev
```

เปิดเว็บเบราว์เซอร์แล้วเข้าไปที่

```bash
http://localhost:3000
```

เมื่อมีการแก้ไขไฟล์ หน้าเว็บจะรีโหลดอัตโนมัติ

---

## เทคโนโลยีที่ใช้

* Next.js
* React
* TypeScript
* Tailwind CSS
* Prisma

---

## โครงสร้างโปรเจกต์

```bash
src/
  app/              # หน้าเว็บและ API routes ของระบบ
  features/         # ฟีเจอร์หลักแยกตามบทบาทผู้ใช้งาน
  services/         # Service สำหรับเรียกใช้งาน API
prisma/             # Prisma schema และไฟล์ที่เกี่ยวข้องกับฐานข้อมูล
public/             # ไฟล์สาธารณะ เช่น รูปภาพและไฟล์อัปโหลด
```

---

## ฟังก์ชันของระบบ

### นักเรียน

* เข้าสู่ระบบด้วยรหัสนักเรียนและรหัสผ่าน
* ดูแดชบอร์ดสรุปข้อมูลส่วนตัว วิชาที่ลงทะเบียน กิจกรรม การเข้าเรียน และคะแนนความประพฤติ
* ดูและจัดการข้อมูลโปรไฟล์นักเรียน รวมถึงข้อมูลห้องเรียนและครูที่ปรึกษา
* ค้นหา ลงทะเบียน เพิ่มรายวิชาลงตะกร้า ยืนยันรายวิชา และดูรายวิชาที่ลงทะเบียนแล้ว
* ดูตารางเรียน ตารางสอบ กิจกรรม และปฏิทินที่เกี่ยวข้อง
* ดูผลการเรียน ผลคะแนนรายวิชา และผลการประเมินด้านการเรียนรู้
* ดูข้อมูลสุขภาพ ประวัติสุขภาพ วัคซีน การตรวจสุขภาพ และรายงานสุขภาพ
* ดูประวัติความประพฤติและคะแนนความประพฤติ
* ทำแบบประเมินกิจกรรม แบบประเมินครู และดูผลการประเมินจากครูที่ปรึกษา

### ครู

* เข้าสู่ระบบด้วยรหัสครูและรหัสผ่าน
* ดูแดชบอร์ดสรุปภาระงาน ตารางสอน ตารางสอบ และข้อมูลนักเรียนที่เกี่ยวข้อง
* ดูและจัดการรายชื่อนักเรียนในความดูแลหรือห้องที่ปรึกษา
* ดูโปรไฟล์นักเรียน อัปโหลด/ลบรูปนักเรียน และบันทึกข้อมูลการประเมินนักเรียน
* บันทึกเวลาเรียนและสถานะการเข้าเรียนของนักเรียนรายวิชา
* จัดการคะแนนรายวิชา หัวข้อคะแนน ตัวชี้วัด หมวดคะแนน และบันทึกคะแนนนักเรียน
* ตัดเกรดและจัดการเกณฑ์การให้ระดับผลการเรียน
* บันทึกและจัดการข้อมูลพฤติกรรมนักเรียน
* บันทึกข้อมูลสมรรถภาพทางกายและจัดการเกณฑ์ประเมินสมรรถภาพ
* จัดการปฏิทินกิจกรรม ตารางสอน ตารางสอบ และภารกิจที่เกี่ยวข้อง
* ดูผลประเมินการสอนและข้อมูลสะท้อนกลับจากนักเรียน

### ผู้อำนวยการ

* เข้าสู่ระบบด้วยรหัสผู้อำนวยการและรหัสผ่าน
* ดูแดชบอร์ดภาพรวมของโรงเรียน เช่น จำนวนนักเรียน ครู กิจกรรม โครงการ และข้อมูลสำคัญ
* จัดการข้อมูลนักเรียน ห้องเรียน ระดับชั้น และจำนวนผู้เรียน
* จัดการข้อมูลครู ตำแหน่งครู และครูที่ปรึกษาประจำห้อง
* จัดการรายวิชา กลุ่มสาระ หมวดวิชา และโครงสร้างหลักสูตร
* จัดการกิจกรรม ปฏิทินกิจกรรม ผู้เกี่ยวข้อง และประเภทกิจกรรม
* จัดการโครงการ แผนงาน ประเภทโครงการ และข้อมูลผู้รับผิดชอบ
* จัดการข้อมูลงบประมาณ รายรับ รายจ่าย หมวดค่าใช้จ่าย และประเภทแหล่งงบประมาณ
* ดูและจัดการข้อมูลพฤติกรรมนักเรียนในภาพรวม
* จัดการแบบประเมิน คำถามประเมิน และผลการประเมินที่เกี่ยวข้องกับโรงเรียน
* ใช้ข้อมูลรายงานเพื่อสนับสนุนการติดตามและตัดสินใจเชิงบริหาร

---

## ความสามารถของระบบ

* รองรับการเข้าสู่ระบบแยกตามบทบาทผู้ใช้งาน
* รองรับการแสดงผลแบบ Responsive
* ใช้ Next.js เพื่อประสิทธิภาพที่ดีในการพัฒนาเว็บแอปพลิเคชัน
* ใช้ TypeScript เพื่อช่วยให้โค้ดมีความชัดเจนและลดข้อผิดพลาด
* ใช้ Prisma สำหรับจัดการโครงสร้างข้อมูลและเชื่อมต่อฐานข้อมูล
