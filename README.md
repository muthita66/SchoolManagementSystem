# Winai School

โปรเจกต์เว็บแอปพลิเคชันที่พัฒนาด้วย Next.js สำหรับระบบ Winai School

---

## 🚀 วิธีเริ่มต้นใช้งาน

ติดตั้ง dependencies ก่อน

```bash id="kz5t8w"
npm install
```

จากนั้นรันโปรเจกต์ด้วยคำสั่ง

```bash id="1c6u4d"
npm run dev
```

เปิดเว็บเบราว์เซอร์แล้วเข้าไปที่

```bash id="o1kcrq"
http://localhost:3000
```

เมื่อมีการแก้ไขไฟล์ หน้าเว็บจะรีโหลดอัตโนมัติ

---

## เทคโนโลยีที่ใช้

* Next.js
* React
* TypeScript
* Tailwind CSS

---

## 📂 โครงสร้างโปรเจกต์

```bash id="tw9j0v"
app/
 ├── page.tsx        # หน้าเว็บไซต์หลัก
 ├── layout.tsx      # Layout หลักของระบบ
 └── globals.css     # ไฟล์ CSS ส่วนกลาง
```

---

## ความสามารถของระบบ

* รองรับการแสดงผลแบบ Responsive
* พัฒนาโดยใช้ Next.js เพื่อประสิทธิภาพที่ดี
* รองรับการพัฒนาร่วมกับ TypeScript
* สามารถ Deploy ได้ง่ายผ่าน Vercel

---

## การ Build สำหรับ Production

```bash id="2j9pgi"
npm run build
npm start
```
